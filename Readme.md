# English to French NMT with Streamlit

This project implements a character-level Sequence-to-Sequence (Seq2Seq) Neural Machine Translation model using TensorFlow/Keras to translate English text into French. It includes a web interface built with Streamlit.

## Dataset
The project uses the English-French sentence pairs from the Tatoeba Project (via `eng_fr.txt`).
